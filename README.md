# private-linter

A simple linter that checks for basic empty line at the end of file.

## Built With

Ruby
Rubocop
Gem
VS Code

## Installing Linter

When you have installed Ruby on your computer, clone the project.
Run linter by typing: bin/main.rb

Enjoy!
